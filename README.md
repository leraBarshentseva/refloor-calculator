# Калькулятор напольных покрытий (Тестовое задание)

Интерактивный калькулятор для расчета стоимости напольных покрытий.

**Ссылка на демо:** [https://lerabarshentseva.github.io/refloor-calculator/](https://lerabarshentseva.github.io/refloor-calculator/)

## Особенности реализации

*   **Архитектура State-First:** Приложение построено на принципах "Single Source of Truth", где UI является отражением объекта состояния.
*   **Модульная структура:** Код разбит на ES-модули для лучшей читаемости и поддержки (`constants.js`, `state.js`, `utils.js`).
*   **Бизнес-логика:** Реализован расчет кратности упаковкам, что обеспечивает честную итоговую цену для клиента.
*   **Продвинутый UX:**
    *   Интерактивная валидация, предотвращающая ввод некорректных данных (например, вычет площади больше доступной).
    *   Плавные CSS-анимации для добавления, удаления и обновления элементов.
    *   Автосохранение данных в `localStorage` при закрытии вкладки.
    *   Адаптивная верстка для корректного отображения на мобильных устройствах.
*   **Качество кода:**
    *   Использование методологии БЭМ для CSS.
    *   Безопасная работа с данными (защита от `JSON.parse` ошибок).
    *   Проект настроен с помощью `Vite`.

## Как запустить локально

1.  Клонировать репозиторий: `git clone https://lerabarshentseva.github.io/refloor-calculator/`
2.  Установить зависимости: `npm install`
3.  Запустить dev-сервер: `npm run dev`
